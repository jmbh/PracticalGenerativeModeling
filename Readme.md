## Practical on Generative Modeling

This practical has the goal of differentiating between generative models and statistical models. Making this distinction will allow us to have an organized discussion about how to use statistical models to learn about the systems we are studying in empirical reserach.

If you are already using Github, you can just clone this repository. If not, you can also download it by clicking on the green button "Code", which gives you a dropdown menu, on which you can click "Download ZIP", to download the entire repository as a ZIP file.
